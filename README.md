# expo-go

To install dependencies:

```bash
bun install
```

To run locally:

```bash
bun ./index.ts url android latest
```

Commands:

```bash
npx expo-go url <ios|android> [sdkVersion|latest]
npx expo-go download <ios|android> [sdkVersion|latest] [outputPath]
```

Build the distributable CLI bundle:

```bash
bun run build
```
